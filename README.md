**MAI** (Motherly AI) 

This emotionally intelligent application allows the User to find  options to give themselves the most well-rounded life with the resources they have.  It is built to emulate the supportive presence of a "Ma" . 

The app helps users explore their emotional health. reflect on experiences, and receive meaningful   responses powered by AI. 

Our quizzes allow the User to get to know themselves on a deeper level, while the AI helps to inform the user about how they can use what they have to support their goals. 

The User can then keep engaging with the chatbot to get more recommendations.
